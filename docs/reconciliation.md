# 月度对账 · 把账里余额和真实账户对齐

每个月底花十分钟，确认衡记里某个账户的余额，和支付宝 / 银行卡 / 现金盒子里**真实的钱**一分不差。对得上，这个月的账就踏实了。

**适用**：开了「商家进阶功能」的小生意用户 / 月底想核一遍账的人。

对账解决一个很现实的问题：你平时记一笔记一笔，难免漏记、记错、或者记重。一个月攒下来，账里写的余额未必等于卡里实际的余额。对账就是拿**对账单的真实余额**做标尺，逐笔勾一遍，把差出来的部分揪出来补平。

> 对账是进阶功能，默认藏起来。要先到 [设置](settings.md) → 打开「商家进阶功能」，左侧栏才会出现「✓ 对账」入口。

---

## 它在哪

对账是**全局入口**，在左侧栏最下方（不属于任何单个账本）。点「✓ 对账」进去，一个页面就能对所有账户——包括跨账本共享的真实资金账户。这一点很关键：你的支付宝既付了货款（生意账本）又买了菜（生活账本），但它只有一个真实余额。对账按**账户**来对，把散在各账本的流水汇到一起。生意账 / 生活账怎么分开记，见 [账本与分账](books.md)。

---

## 对一个账户，五步

1. **选账户。** 顶部「对账账户」下拉，列出所有资产 / 负债账户。每项后面会标它属于哪个账本（或「全局共享」），以及还剩几笔「待核销」。先挑一个，比如「支付宝」。
2. **填对账单余额。** 打开支付宝 App 看当前余额，照抄进「对账单余额（元）」。负债账户（比如信用卡欠款）记负数。
3. **填对账截止日。** 默认今天，一般不用改。
4. **逐笔勾对。** 下面「流水勾对」列出这个账户的每一笔流水，按日期排好。对着对账单一笔笔勾——账里有、对账单上也有的，打勾。
5. **看差额，对到 0，完成。** 页面底部实时显示「已勾选合计」和「差额」。差额＝对账单余额 −  已勾选合计。差额变成 **0**（会变绿）时，点「完成对账」即可。

完成后，这次勾上的流水会被标成「已核销」，下次对账就不用再碰它们，列表只盯新增的笔数。

---

## 差额对不平怎么办

差额 ≠ 0，说明账和现实有出入。常见三种情况，都能在对账页当场解决，不用跳走丢掉勾选进度：

### 1. 对账单上有、账里漏记了 → 补录一笔

点流水列表下面的「＋ 补录一笔」。填好归属账本、类型（收 / 支）、金额、分类、日期，点「补录并勾选」。这笔会立刻记进账、并自动打勾，差额随之收窄。

> 全局共享账户补录时可以选落到哪个账本（买菜归生活账本、进货归生意账本）；账本专属账户只能落它自己的账本。

### 2. 账里有、记重了或记错了 → 删掉

每一行流水右侧有个「×」，点它删掉整笔交易（连同它的对方分录一起删）。如果这笔已经核销过，会多一道确认提醒——删它会影响之前完成的对账记录。**删除不可撤销**，看清楚再删。

### 3. 实在查不出差在哪 → 一键调平（逃生口）

有时候就是差几毛钱，翻来覆去找不到。差额 ≠ 0 时底部会出现一个「记盘盈盘亏调整 ××」按钮，点它：衡记自动记一笔「盘盈盘亏」调整，刚好把差额补平，并自动勾上。差额归零，就能完成对账了。

这是体面的收尾，不是糊弄——做生意现金账难免有零头出入，「盘盈盘亏」就是会计里专门吸收这种小差的科目。但如果差的是一笔大钱，别急着调平，先回[流水](recording.md)里翻翻是不是漏了哪笔大的。

## 一个账户横跨多个账本时

如果选的是全局共享账户（比如支付宝同时服务生意账本和生活账本），流水勾对区上方会出现一排账本筛选标签：「全部账本」「日常生活」「小卖部」……

- 这些筛选**只影响你看到哪些流水**，不影响整账户的对账——差额永远按这个账户的**全部**流水算。
- 点某个账本标签，标题会显示「本账本已勾选 ××」，方便你分别核对每个账本贡献了多少。
- 每一行流水也会标上它属于哪个账本，一眼看清这笔是生意的还是生活的。

账户为什么能全局共享、怎么设，见 [账户](accounts.md)。

## 设个对账提醒，省得忘

到 [设置](settings.md) → 「对账提醒」可以定一个每月对账日：

| 选项 | 说明 |
| --- | --- |
| 对账日 | 关闭提醒 / 每月最后一天 / 每月 1–28 日 |
| 提前提醒 | 当天 / 提前 1、2、3、5、7 天 |

设好之后，临近对账日时，会在账本顶部弹一条提醒，一键「去对账」。贴心的是：**某账本里的账户已经全部核销，就不会再打扰你**——已经对完的账本不啰嗦。

---

## 原理：勾对背后发生了什么

你只是在打勾，但底层是一套严谨的复式记账在跑：

- 「记一笔」时，每笔流水都生成了一对借贷分录（钱从哪来、到哪去），账本天然平衡。
- 「核销（cleared）」是给账户的某条分录贴的一个标记，意思是「这笔已经和真实账户对上了」。
- **对账单余额 − 已勾选分录之和 = 差额**。差额为 0，就证明你账里被核销的这些流水，加起来正好等于真实账户的余额。
- 完成对账时，勾上的标为已核销、没勾的标为未核销，并存一条对账记录（账户 + 对账单余额 + 日期），作为下次对账的基线。

你不用懂这些也能用——勾、看差额、完成，就够了。
