# 账本与分账：把生意账和生活账分清楚

衡记的核心，是让你用一个软件同时记「家里的账」和「店里的账」，又不把两边搅成一锅粥。

**适用**：摆摊、开小店、做小网店、搞社区团购的个体户和小老板——尤其是「一个支付宝既付货款又买菜」、生意账和生活账长期混在一起的人。

## 一句话理解

- **账本**＝一本独立的账。家里一本、店里一本，互不干扰。
- **真实钱包**（现金、银行卡、支付宝、微信）可以设成「全局共享」，被多本账共用——因为现实里它们本来就是同一个账户。

钱是同一笔钱，但「这笔属于生意还是生活」分得清清楚。报税、算店里赚多少，再也不用对着一堆混在一起的流水发愁。

## 三类账本

新建账本时选一个类型，衡记会按类型给你配好一套开箱即用的科目（收支分类）：

| 类型 | 给谁用 | 大致内容 |
| --- | --- | --- |
| 👤 个人 | 记家庭/个人日常收支 | 现金、银行卡、支付宝、微信钱包、信用卡、花呗；工资、餐饮、交通、购物、居住等 |
| 💼 生意 | 记店铺/生意经营 | 对公账户、微信商户、支付宝商户、现金、应收账款；营业收入、进货成本、运费杂费等 |
| 📈 投资 | 单独跟踪一笔投资 | 投资账户、投资盈亏、期初余额 |

> **投资账本默认看不到。** 它属于进阶功能，要先到 [设置](settings.md) 里「开启商家进阶功能」，新建账本时才会多出「投资」这个选项。普通用户日常用「个人」和「生意」两类就够了。

同一个类型可以建多本。比如夫妻俩各记各的，就建两本「个人」账本（「我的日常」「妻子的账本」）；开了两家店，就建两本「生意」账本。

## 新建、切换、归档账本

账本都在左侧栏「账本」一栏里。

**新建一本账：**

1. 点左侧栏的「＋ 新建账本」。
2. 填账本名称（如「我的小店」）。
3. 选类型：个人 / 生意（开了进阶功能才有投资）。
4. 点「创建」。衡记会自动配好这类账本的默认科目，并直接进入新账本。

**切换账本：** 在左侧栏直接点账本名即可。当前账本会高亮，主界面随之切换。

**归档账本：** 进入要归档的账本，点右上角「归档账本」并确认。归档后：

- 这本账从左侧栏列表隐藏，[财务总表](#财务总表全部账本一屏看清) 也不再把它计入。
- **数据不会删除**，只是收起来。
- 你的共享钱包（全局账户）即使是在被归档的账本里建的，也仍然保留，不会从别的账本里消失。

## 英雄场景：钱是同一个钱包，账分到两本

这是衡记为个体户解决的最核心痛点。

假设你有一个支付宝，既给店里付货款、也给家里买菜。过去记账，要么生意账生活账混在一起算不清，要么得人为拆成两个钱包、很别扭。

衡记的做法：**把这个支付宝设成「全局共享」账户**，然后——

- 在「我的日常」账本里记买菜、付房租；
- 在「我的小店」账本里记付货款、收客户的钱；
- 两本账都能选到同一个支付宝下账。

于是每一笔都明确归属到「生意」或「生活」，但钱包余额始终是这一个真实支付宝的真实余额，不用你手动对。设成共享账户后，它在左侧栏账户名旁会标一个「全局」小标记。

怎么把一个账户设成全局共享、哪些账户能设（只有真金白银的钱包能设，应收账款、库存这种「虚拟科目」不行），详见 [账户](accounts.md)。

### 原理（点到为止）

衡记底层是严谨的复式记账，但你只管「记一笔」，借贷分录是软件自动生成的。这里只需知道一件事：

- **真实钱包（现金/银行/支付宝/微信）的余额属于「全局资金」**，不算进任何单独一本账的「净资产」——因为这笔钱是几本账共用的，算进谁都会重复。
- **每本账自己算的是「经营/生活」那部分**：本月赚了多少、花了多少、店里有多少应收和库存。

所以在 [财务总表](#财务总表全部账本一屏看清) 上你会看到「全局资金」单独列一行，各账本则只显示自己专属的那部分，加起来不重复、不打架。

## 财务总表：全部账本一屏看清

左侧栏最上方的「🧮 财务总表」是所有账本的汇总入口（也是打开衡记的默认落地页）。它一屏给你：

- **全部净资产**：全局共享资金 + 各账本专属净额之和，几本账加起来的家底，不重复计算。
- **本月总收入 / 总支出 / 总结余**：所有账本合起来的本月进出。
- **各账本卡片**：每本账一张卡，显示它的净额和本月结余（生意账本显示「经营净额」和「本月利润」，个人账本显示「专属净额」和「本月结余」）。点卡片直接进入那本账。

如果你持有外币账户，净资产还会按币种分别列出小计，再给一个折合总值。多币种细节见 [账户](accounts.md)。

## 各类账本的默认科目差异

新建账本时，衡记按类型自动配好一套科目，省得你从零搭：

- **个人**：纯生活收支——现金/银行卡/支付宝/微信钱包/信用卡/花呗，加工资、餐饮、交通、购物、居住、娱乐、医疗等分类。没有任何生意元素。
- **生意**：经营科目——对公账户/微信商户/支付宝商户/现金/应收账款，加营业收入、进货成本、运费杂费等。库存、应付账款这类科目，等你真用到进货、赊账时由系统按需自动建，不会一上来堆一堆看不懂的科目。
- **投资**：最简——投资账户、投资盈亏、期初余额，专门跟踪一笔投资的现值涨跌。

科目不是写死的，可以按自己的习惯改名、增删，做法见 [账户](accounts.md)。
