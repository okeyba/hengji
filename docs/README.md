# 衡记 Héng · 使用手册

**中文** · [English](en/README.md) ·  [← 返回项目首页](../README.md)

按功能分篇。普通用户看「入门」几篇就够；开了商家进阶功能再看「进阶」。

## 入门（普通用户 / 个体户）

- [账本与分账](books.md) — 个人 / 生意 / 投资多账本，怎么把生意账和生活账分开。
- [账户与全局共享](accounts.md) — 账户怎么建、真实钱包怎么在多账本之间共用。
- [记一笔与流水](recording.md) — 记支出 / 收入 / 转账，查看和管理流水。
- [总览·财务总表·预算](overview.md) — 看本账本和全部身家，设月度预算。
- [简单与专业模式](modes.md) — 极简模式默认，什么时候开「商家进阶功能」。

## 进阶（商家 / 小生意，需开启「商家进阶功能」）

- [商品与库存](inventory.md) — 商品主数据、移动加权均价、进货、盘点。
- [订单·客户·应收](orders.md) — 开单、完成确认收入、赊账分期收款、账龄到期。
- [采购·供应商·应付](purchases.md) — 采购单、供应商档案、应付与还款。
- [额外费用与公式引擎](fees.md) — 佣金 / 运费等可复用阶梯费用，开单按行勾选。
- [月度对账](reconciliation.md) — 逐笔勾对、盘盈盘亏、内联补录。
- [多币种](multi-currency.md) — 自管币种、汇率、展示币种、换汇录入。

## 设置与开发

- [设置（记账口径·提醒·进阶）](settings.md) — 全局设置、权责发生制 / 收付实现制。
- [开发者：构建·架构·贡献](development.md) — 从源码构建、跑测试、打安装包。
