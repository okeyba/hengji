# 记一笔与流水

把每一笔进出账都记下来，是衡记最常用的动作。点几下、填个金额，就完事了。

**适用**：所有用衡记的人 / 日常收钱、付钱、在自己几个账户之间倒钱的时候。

---

## 记一笔：三种类型

打开「记一笔」，最上面有三个标签：**支出**、**收入**、**转账**。先选对类型，再往下填。

- **支出**——钱出去了。进货、买菜、付房租、给员工发工资。
- **收入**——钱进来了。卖货收款、平台打款、别人还你钱。
- **转账**——钱没多也没少，只是从你的一个账户挪到另一个。比如从支付宝提现到银行卡、往微信钱包充钱。转账不算赚也不算花，别拿它记收入或支出。

### 支出 / 收入 怎么填

1. 选 **支出** 或 **收入**。
2. 填 **金额**（只填正数，比如 `38.50`）。
3. 选 **日期**（默认今天，点一下能改）。
4. 选 **分类**（支出）或 **来源**（收入）——比如餐饮、交通、进货成本；收入则是营业收入、工资等。
5. 选 **账户**——这笔钱是从哪个真实账户出/进的，比如「微信」「招行卡」。
6. **备注**（可选）——填商家名或一句话，方便以后认出这笔是什么。
7. 点 **保存**。看到「已记账 ✓」就成了。

> 金额栏标题里的符号（如 `¥`）会跟着你选的账户币种变。详见下面的「换汇」。

### 转账 怎么填

1. 选 **转账**。
2. 填 **金额**。
3. 选 **从**（哪个账户出钱）和 **到**（哪个账户进钱）——两个不能是同一个，否则会提示「转出与转入账户不能相同」。
4. 备注可选，点 **保存**。

---

## 分类、账户，从哪来？

「记一笔」里能选的分类和账户，都是你提前建好的：

- 真实的钱放在哪、欠了谁的钱——在 [账户](accounts.md) 里管理，全衡记共享。
- 这笔记到哪个账本（生意账还是生活账）——看你当前打开的是哪个账本，记一笔会自动归到它名下。怎么分账见 [账本与分账](books.md)。

所以如果下拉里找不到想要的账户或分类，先去对应页面把它建出来。

---

## 多币种：换汇怎么记

如果你只用人民币，跳过这节。

当你做一笔 **转账**，而且「从」和「到」两个账户的币种不一样时，衡记自动认出这是 **换汇**，表单会多出一栏：

- **汇出（原币）**——你实际付出去多少（按转出账户的币种）。
- **到账（原币）**——对方实际到账多少（按转入账户的币种）。

衡记只记你填的这两个真实数字，**不会**自己去乘汇率、也不会替你算汇兑损益。两边金额差出来的那点，就藏在两个账户各自的原币余额里，只有在把全部资产折算成人民币看总值时才会体现出来。两栏都得填正数，少填会提示「请输入有效的到账金额」。

---

## 原理：背后悄悄记了一笔复式账

你看到的是「记一笔」，衡记在底层其实把它拆成了 **两条对冲的分录**，让账永远是平的——这就是复式记账，但你完全不用懂它也能用。

| 你记的 | 底层发生了什么 |
| --- | --- |
| 支出 100 | 「分类」科目 +100，「账户」−100 |
| 收入 100 | 「账户」+100，「来源」科目 −100 |
| 转账 100 | 「到」账户 +100，「从」账户 −100 |

普通（单币种）交易，两条分录之和必须等于 0，否则衡记不让保存——这是它帮你守住的底线：钱不会凭空多出来或消失。换汇是例外：两条腿币种不同、本就不等值，这条「求和为 0」就豁免了。

这层细节平时你看不到也不用管，知道「记一笔背后是笔平账」就够了。

## 流水页：查看你记过的每一笔

左侧菜单的 **流水**，按时间倒序列出你记过的所有交易。每行长这样：

- 左边一个 **图标**（按分类自动配，比如餐饮 🍜、转账 🔁、换汇 💱）。
- 中间是 **标题 + 一行小字**：标题优先显示你填的备注，小字是「分类 · 账户 · 日期」。
- 右边是 **金额**：支出显示为负、收入为正、转账中性。
- 如果这笔属于生意账，标题旁会挂一个灰色 **「生意」** 小标。

页面顶部会显示当前账本名和「共 N 笔」。

### 流水只看当前账本

流水页 **只显示你当前打开的那个账本** 的交易。想看另一个账本（比如从「生活」切到「生意」），先在顶部切换账本，流水会跟着换。换句话说，生意账和生活账各看各的，不会混在一张单子里。

### 删除一笔

每行最右边有个 **×**，点它删除。会弹一个确认框：

- 普通交易——问你「删除这笔交易？」，确认即删。
- 如果这笔含 **已核销** 的分录，提示会变成「这笔交易含已核销分录，删除会打散已完成的对账（届时需重新对账）」。意思是你之前对过账、把它跟银行账单核对上了，删掉就得重新核对。看 [对账](reconciliation.md) 了解核销是怎么回事。

删除是软删除，记错了也别慌，但目前界面里删了就从流水消失，记账时仔细点。
