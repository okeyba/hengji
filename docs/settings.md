# 设置（记账口径 · 提醒 · 进阶）

设置页放的是「全局开关」——改一次，应用到你所有的账本，而不是只对当前这本生效。

**适用**：所有用户做基础设置；开了「商家进阶功能」后还会用到记账口径、对账 / 往来提醒、多币种这些进阶项。

---

## 全局：改一次，所有账本一起变

打开「设置」，最上面写着「全局 · 应用于所有账本」。这一页里的每一项，无论你当前在看哪本账，都是统一生效的——比如你把「展示币种」设成美元，那总览和每一本账的净资产都按美元折算显示。不需要逐本去设。

## 极简模式 vs 商家进阶功能

衡记默认是**极简模式**：普通用户打开只看到 总览 / 流水 / 预算 / 账户 四样，够记日常小账。专业功能藏起来，界面更清爽。

设置页第一张卡片就是那个总开关：

> ☐ **开启商家进阶功能**

打开后才解锁这些：进销存（商品 / 库存 / 采购 / 供应商 / 应收应付）、额外费用、月度对账、多币种、记账口径切换、投资账本。

**所以**：本篇下面讲的「记账口径」「对账提醒」「往来到期提醒」「多币种」四张卡片，都要先勾上这个开关才会出现。只记日常小账的话保持关闭即可，随时回来打开。

## 记账口径：权责发生制 / 收付实现制

> 仅当开启了「商家进阶功能」时出现。

这个开关只改「本月收入 / 利润」这类报表数字怎么算，**不动你已经记的任何一笔账**。底层永远按权责发生制记，这里切的只是报表的呈现口径。

| 口径 | 一句话 | 适合 |
| --- | --- | --- |
| **权责发生制**（默认） | 订单完成就算收入，哪怕钱还没收（赊账也计） | 想要正规、贴资产负债表 |
| **收付实现制** | 只有钱实际到账才算收入，赊账等收到才计 | 想看真实现金流、直观 |

### 什么时候两者会不一样？

**只有「有赊账的生意账本」两个口径才会差**。如果你做的是一手交钱一手交货的现金生意，或者是个人账、投资账，两种口径算出来完全一样——不用纠结，留默认就行。

### 原理（点到为止）

差异只来自应收账款（AR）的变化：

```
收付实现制收入 = 权责发生制收入 − 本期应收净增（ΔAR）
```

- 赊出去一笔货：权责口径当期就计收入，现金口径要等回款才计；
- 客户后来还钱：现金口径这时才把它算进收入。

底层那两笔分录一字不改，衡记只是在出报表时换了种加总方式。赊销、账期这些概念见 [账本与分账](books.md) 和 [应收应付](orders.md)。

## 对账提醒

> 仅当开启了「商家进阶功能」时出现。

设一个「每月对账日」，临近时衡记会在账本顶部提醒你去「对账」页核对账面和实际余额对不对得上。

1. **对账日**：选「每月最后一天」，或「每月 N 日」（1–28 任选），或「关闭提醒」。
2. **提前提醒**：选了对账日后才出现，可选「当天」或提前 1 / 2 / 3 / 5 / 7 天（默认提前 3 天）。

贴心处：**某本账已经全部核销完了，就不会再拿对账提醒打扰你**。怎么核销、对账页怎么用，见 [对账](reconciliation.md)。

## 往来到期提醒

> 仅当开启了「商家进阶功能」时出现。

生意账里有人欠你钱（应收）或你欠供应商钱（应付）时，到期前 / 逾期后衡记在账本顶部提醒你去跟进收款 / 安排付款。

- **应收到期日** = 客户下单日 + 账期；
- **应付到期日** = 采购日 + 供应商账期。

只有一个选项「提前提醒」：

| 选项 | 含义 |
| --- | --- |
| 关闭提醒 | 不提醒 |
| 到期当天 | 当天才提醒 |
| 提前 3 / 7 / 14 / 30 天 | 提前几天开始提醒 |

默认提前 7 天（应收逾期是真痛点，所以默认开着）。**没有往来、或全部结清了就不打扰**。账期怎么设、应收应付怎么记，见 [应收应付](orders.md)。

## 多币种

> 仅当开启了「商家进阶功能」时出现。详细玩法见 [多币种](multi-currency.md)。

只用人民币的话，这个开关保持关闭——账户、报表里完全不出现币种选项，干净。

需要记美元 / 日元 / 比特币等，勾上「开启多币种」，之后建账户时就能选币种。开了之后这里还能：

- **展示币种**：总表和各账本的净资产、收支按你选的币种折算显示（各账户余额和流水仍按自己的原币显示，不变）。
- **自定义币种**：填代码 / 符号 / 名称 / 小数位 / 对人民币汇率。人民币是本位币，固定 ¥、汇率 1，不可改。

两个安全提醒：

1. **已经有外币账户在用时，多币种开关会被锁定为开**（关掉却还显示着外币就自相矛盾了）。要回到纯人民币，先去「账户」页把所有外币账户归档。
2. **某币种已有账户在用时，它的小数位不能再改**（改了会错读已记金额），也不能删除（先归档 / 改用别币种的账户）。

汇率怎么折算、财务总表怎么按币种分组，展开讲在 [多币种](multi-currency.md)。

## 数据存在哪？

所有设置和账目都存在你这台电脑的本地 SQLite 文件里，不联网、不上传。换台电脑或重装前，记得自己复制一份这个数据文件做备份。
