# 简单与专业：两套界面，按你的生意大小切换

刚打开衡记，你看到的是「简单模式」——只有总览、流水、预算、账户四块，记一笔生意账跟记买菜钱一样轻。等生意做大、要管进货和赊账了，再到设置里打开「商家进阶功能」，进销存、对账、多币种这些专业工具才会一次性冒出来。

**适用**：所有用户 / 不确定该不该打开那一堆专业功能时。

---

## 默认就是简单模式

衡记开箱即用，**默认不显示任何专业功能**。无论你建的是个人账本还是[生意账本](books.md)，左侧标签都只有这四个：

| 标签 | 干什么 |
|------|--------|
| 总览 | 这本账的收入、支出、利润一眼看完 |
| 流水 | 一笔一笔记账、翻账（[怎么记一笔](recording.md)） |
| 预算 | 给某类开销设上限，超了提醒 |
| 账户 | 管你的支付宝、微信、现金、银行卡（[账户](accounts.md)） |

> **个体户为什么默认这样？**
> 摆摊、开小店、做社区团购的人，最常见的痛点是**生意账和生活账混在一个支付宝里**——同一个码既收货款又付菜钱。衡记的核心是帮你用[多账本](books.md)把这两本账分清楚，而不是逼你学一套进销存软件。所以默认只留「记一笔」最需要的四块，其余的等你真用得上再说。

简单模式下，生意账本和个人账本的界面**几乎一模一样**。区别只在账本类型本身（影响科目和总表归类），不在于你眼前多了多少按钮。

---

## 什么时候该开「商家进阶功能」

如果你开始遇到下面这些情况，就该打开了：

- 要管**进货、库存、卖货**，知道每件商品还剩多少、赚多少；
- 客户**先拿货后给钱**（赊账），你想盯着谁还欠你钱、什么时候到期；
- 进货也是**先欠着供应商**，要记账期、防漏付；
- 一笔订单除了货款还有**运费、包装、平台抽成**这类额外费用；
- 想每月**对一次账**，核对软件里的余额和支付宝实际余额对不对得上；
- 收支里掺了**美元、日元甚至比特币**等外币；
- 你还想单独建一本**投资账本**记股票基金。

只记日常小账、不碰这些的话，**保持关闭，界面更清爽**——这也是衡记给个体户的默认建议。

---

## 怎么打开（一步）

1. 点左侧最下方的「⚙ 设置」。
2. 最上面第一项就是 **「开启商家进阶功能」**，勾上它。
3. 立刻生效，无需重启。下面会多出一排设置卡片，左侧生意账本也会长出新标签。

想关回简单模式，把这个勾去掉即可，**你已经记下的数据一条都不会丢**——只是那些专业标签暂时藏起来。

---

## 打开后解锁了什么

勾上「开启商家进阶功能」后，这些一次性全部出现：

### 1. 进销存（生意账本多出 6 个标签）

生意账本的标签从 4 个扩成完整一套，按生意流程排列：

| 新增标签 | 作用 |
|----------|------|
| 订单 | 记每一单卖了什么、收没收钱、客户欠多少 |
| 客户 | 管客户和他们的账期（应收） |
| 供应商 | 管供应商和进货账期（应付） |
| 商品 | 你卖的东西的清单 |
| 库存 | 每件商品还剩多少 |
| 采购 | 记进货 |

（原有的总览 / 流水 / 预算 / 账户仍在，只是排到后面。）

### 2. 额外费用（「费用」标签）

给订单挂运费、包装费、平台抽成这类附加费用，可以预设档位重复用（金额也支持负数，用来记满减 / 折扣）。

### 3. 月度对账

左侧会多出一个「✓ 对账」入口，定期核对软件记的账户余额和真实余额对不对得上。详见[对账](reconciliation.md)。

### 4. 多币种与记账口径（在设置里）

设置页会展开几张新卡片：

- **记账口径**：切换「本月收入 / 利润」按**权责发生制**（订单完成就算收入，含赊账）还是**收付实现制**（钱到账才算）来呈现。注意这只改报表怎么看，**不改动你已经记的任何一笔**，底层永远按权责发生制记账。只有带赊账的生意账本两种口径才有差别，个人 / 投资账本结果一样。
- **多币种**：打开后可添加美元 / 日元 / 比特币等币种，建账户时选币种，并指定一个「展示币种」把总表折算显示。（只要你建过外币账户，多币种会自动锁定为开，需先归档所有外币账户才能关回纯人民币。）

### 5. 投资账本类型

新建账本时，账本类型下拉里会多出「投资」一项，用来单独记股票、基金等。

### 6. 提醒横幅

进阶模式下，账本顶部会在合适的时候弹出提醒条：

- **对账提醒**：到了你设定的每月对账日附近，且该账本还有没核对的流水时；
- **应收到期提醒**：有客户的赊账快到期或已逾期，提醒你去收款；
- **应付到期提醒**：你欠供应商的货款快到期，提醒你去付款。

这些提醒在简单模式下**完全不出现**——既不打扰，也因为简单模式里压根没有对账、订单这些页面可去。提醒的对账日、提前天数都在设置里自己调。

---

## 原理：界面在变，账没变

简单和专业只是**两套界面外壳**。无论开关怎么切：

- 你记的每一笔，背后都是衡记自动生成的标准复式分录，严谨程度不打折；
- 切换模式只是显示 / 隐藏标签和设置项，**从不删改数据**；
- 关掉进阶功能再打开，之前记的订单、库存、外币账户原样还在。

所以你可以放心地先用简单模式起步，生意做大了再随时打开专业工具——衡记一直在底层替你把账记对。
