# 总览 · 财务总表 · 预算

看懂自己的钱：每个账本一页「总览」,把全部账本拢成一张「财务总表」,再用「预算」给自己定个花钱的红线。

**适用**:所有人 / 想一眼知道「这个月赚没赚到、钱还剩多少、有没有花超」的场景。

衡记把生意账和生活账分在不同[账本](books.md)里,但它们常常共用同一个真实账户(一个支付宝既付货款又买菜)。这一篇讲的三块——账本总览、跨账本的财务总表、预算——就是帮你在「分开看」和「合起来看」之间随时切换。

---

## 一、账本总览:这个账本现在怎么样

打开任意一个账本,左侧默认停在「总览」。它从上到下分三块。

### 上方一排数字卡

最大的一张卡是这个账本的**净额**:

- 生活类账本叫「本账本净额」——这个账本里专属账户的资产减负债。
- 生意类账本叫「经营净额」——只算这门生意自己的家当(进货、设备、欠款等),不把你的私人钱包算进来。

紧挨着可能有一张「**可用资金 · 全局共享**」。这是你设为「全局」的真实账户(支付宝、银行卡这类两边都在用的钱)的合计。它单独列、不并进上面的净额里,这样你既能看清「这门生意本身值多少」,又能看到「手头实际能动的现金有多少」。如果你没有设过全局账户,这张卡不出现。

> 为什么要分开?因为公用的支付宝余额属于「你这个人」,不属于某一门生意。把它单列,生意的盈亏才不会被生活开销搅浑。账户怎么设全局,见[账户](accounts.md)。

后面三张是本月的流量:

| 卡片 | 含义 |
| --- | --- |
| 本月收入 | 本月所有收入合计 |
| 本月支出 | 本月所有支出合计 |
| 本月结余 / 本月利润 | 收入减支出;生活账本叫「结余」,生意账本叫「利润」 |

页面右上角还会显示当前月份。生意账本若开了收付实现制,会带一个「收付实现制」小标;若做过对账,会显示「本期已对账 ✓」或「N 个账户待对账」。

### 中间:资产分布饼图 + 记一笔

左边是**资产分布**饼图,把这个账本专属的各个资产账户按金额画成一圈,旁边列出每块的名字和余额,标题处给出合计。注意:全局共享账户(那笔「可用资金")不画进这个饼,饼里只有这个账本自己的资产。

右边是「记一笔」的快速入口,日常记账从这里开始。

### 下方:最近交易

列出这个账本最近的 6 笔交易。还没记过账时,会提示「还没有交易,从『记一笔』开始」。想看全部,去[流水](recording.md)。

## 二、财务总表:把所有账本合起来看

左侧账本列表最上面有一个「🧮 财务总表」。点它,就跳出单个账本,看你名下**全部账本的总和**。

最大的一张卡是「**全部净资产**」,标题里会写「(N 个账本汇总)」。它的算法很直白:

```
全部净资产 = 全局资金(共享) + 各账本各自的专属净额之和
```

下面同样有「**全局资金 · 全账本共享**」单列一次(那些公用的支付宝、银行卡),以及本月总收入 / 总支出 / 总结余。

> 全局资金只算一次。它不归任何一个账本,所以无论你有几个账本共用同一个支付宝,它都只被计入「全局资金」这一格,绝不重复累加。这正是衡记「混账分清」的关键。

再往下是每个账本一张**卡片**,显示这个账本的名字、类型、净额,以及本月结余/利润和交易笔数。**点任意一张卡片,就进入那个账本的总览**,等于一个快速跳转。

> 已归档的账本不计入财务总表;但你设为全局的账户始终参与统计,即使它最初挂的那个账本被归档了也不受影响。

## 三、预算:给每个花钱的分类定月度红线

在生活类账本(或开了进阶功能前的生意账本)里,左侧有「预算」。它帮你按**支出分类**设一个**每月**的上限,然后用进度条告诉你花到哪一步了。

### 加一个预算

1. 进入账本的「预算」页,看下半部分「新增预算」。
2. 「分类」下拉里选一个支出分类(比如「餐饮」「房租」)。下拉里只会列出**还没设过预算的支出分类**,设过的不再重复出现。
3. 在「月度限额(元)」里填金额,比如 `1000`。
4. 点「添加」。没选分类或金额无效时,会提示「请选择分类并输入有效限额」。

### 看用量

每条预算是一行,显示「已花 / 限额」和一条进度条:

- 没超时进度条按比例填充(最多 100%)。
- 花超了,数字会标红并加「· 已超支」,进度条也变色。

想删掉某条预算,点那行右边的「×」,确认即可。

### 原理(点到为止)

- 预算只针对**支出分类**,不针对真实账户——它管的是「这类钱花了多少」,不是「某个钱包还剩多少」。
- 「已花」就是本月该分类下所有记账的合计,你正常[记一笔](recording.md)时它自动累加,不用手动更新。
- 预算一律按**人民币(¥)**计。如果你记了外币支出,会按当前汇率折算后计入(展示币种就算选的不是人民币,预算口径也仍是人民币)。
